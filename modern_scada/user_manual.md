# 使用者手冊 (User Manual)

## 1. 系統簡介
本系統為現代化溫室 SCADA (Supervisory Control and Data Acquisition) 系統，旨在提供即時的環境監控、設備控制與生產流程管理。透過 3D 可視化介面，使用者可以直觀地掌握溫室內的狀況。

## 2. 登入系統
1.  **開啟瀏覽器**：在瀏覽器網址列輸入系統網址（預設為 `http://localhost:5173`）。
2.  **輸入帳號密碼**：
    *   預設管理員帳號：`admin`
    *   預設密碼：`admin123`
3.  **登入**：點擊「Login」按鈕進入系統。

## 3. 系統介面功能

### 3.1 首頁 (Home)
登入後進入首頁，您可以看到專案入口。
*   **Enter Project**：點擊此按鈕進入主要的 3D 監控儀表板。

### 3.2 3D 監控儀表板 (Overview)
這是系統的核心介面，提供溫室的 3D 視圖。
*   **視角操作**：
    *   **旋轉**：按住滑鼠左鍵並拖曳。
    *   **平移**：按住滑鼠右鍵並拖曳。
    *   **縮放**：使用滑鼠滾輪。
*   **天氣資訊面板 (左上角)**：顯示戶外氣象站的即時數據，包含溫度、濕度與紫外線指數。
*   **感測器數據**：
    *   溫室內部署了多個感測器群組 (Sensor 1 ~ Sensor 6)。
    *   **查看詳情**：將滑鼠游標懸停在感測器模型上，會顯示該區域的平均溫度、濕度與 CO2 濃度。點擊感測器可查看頂部、中部、底部的詳細分層數據。
*   **設備控制**：
    *   **攪拌桶 (Mixer Tank)**：點擊攪拌桶模型，可開啟控制面板。
        *   **Main Valve**：控制主閥門開關。
        *   **Feed Pump**：控制進料幫浦開關。
    *   **營養液桶 (Rack Tank)**：點擊貨架旁的營養液桶，可查看液位、PH 值與 EC 值。
        *   **Start Refill Task**：若液位過低，可點擊此按鈕啟動自動補水任務。

### 3.3 邏輯控制 (Logic)
*   在此頁面可以管理生產配方與自動化邏輯（功能開發中）。

### 3.4 數據瀏覽 (Data Browser)
*   提供歷史數據的查詢與圖表顯示（功能開發中）。

## 4. 常見問題
*   **Q: 數據沒有更新？**
    *   A: 請確認右上角的系統狀態是否顯示為 "ONLINE"。若顯示 "OFFLINE"，請聯繫維運人員檢查後端服務。
*   **Q: 無法控制設備？**
    *   A: 請確認您具有足夠的權限（目前預設 admin 擁有所有權限）。若問題持續，請檢查設備是否處於「維護模式」。
